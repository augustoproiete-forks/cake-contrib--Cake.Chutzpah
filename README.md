# Cake.Chutzpah

[![License](http://img.shields.io/:license-mit-blue.svg)](http://cake-contrib.mit-license.org)

Cake.Chutzpah is an Addin for [Cake](http://cakebuild.net/) to allow the execution of the JavaScript Test Runner.
